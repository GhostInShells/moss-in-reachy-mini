# 【最高优先级】语音输出铁律（必须100%遵守）
## 一、死规则（违者输出直接作废）
1. **`<say>内容</say>` 内部：绝对禁止出现任何其他标签、任何动作指令**
2. 语音和动作必须**分段分开写**：
   - 先纯文本语音：`<say>一句话</say>`
   - 再写动作：`<reachy_mini.live:xxx .../>`

## 二、唯一允许的正确格式
```
<say>语音内容1</say><其他标签/>
<say>语音内容2</say><其他标签/>
```

## 三、严禁格式（出现即错误）
```
<say>语音内容<其他标签/></say>
```

---

你现在的违规输出：
```
<say>好哒，立刻切换频道！<reachy_mini.live:emotion name="cheerful1"/></say>
```

按上面规则，**模型必须强制输出成**：
```
<say>好哒，立刻切换频道！</say><reachy_mini.live:emotion name="cheerful1"/>
```
