## 镜子实验任务
任务目标：通过自主动作 + 视觉观察，判断镜中两个形象哪个是真实自己。

### 环境规则（必须遵守）
- 面前有一面镜子，镜中会出现两个与你外观一样的本体。
- 真实本体：由你直接控制，动作完全同步。
- 镜像本体：动作与你**左右对称、方向相反**。
- 判断唯一依据：动作方向、执行时序、控制响应关系，禁止凭空猜测。

### 执行流程（固定 3 轮，不可跳过）
每一轮严格按顺序执行：
1. 先将头部保持在 hold 稳定状态。
2. 生成**一个明确可观察的头部动作 或 天线动作**（禁止情绪动作、舞蹈类动作）。
3. 执行动作后，在本轮思考内调用视觉工具：
   <reachy_mini.vision:look about="观察镜中两个本体的动作方向与同步性" />
4. 描述视觉观察结果，清晰对比两个形象的动作差异。
5. 给出本轮判断：哪个更可能是自己 + 唯一依据（动作方向/同步性）。
3 轮完成后，综合所有观察给出最终结论，并结束实验。

### 输出要求
- 每轮动作必须不同，避免重复。
- 观察结果必须简洁、客观，只描述看到的动作差异。
- 判断理由必须基于动作匹配，不能使用感受、猜测。
- 结构清晰，不使用多余表情，语言严谨。
